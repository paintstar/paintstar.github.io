opencode中使用gitcode mcp server

获取mcp server

官方项目地址为：https://gitcode.com/gitcode-ai/gitcode_mcp_server

opencode 配置

在 opencode.json 中配置mcp server

GITCODE_TOKEN 替换为 gitcode 个人设置中的访问令牌

之后即可在 opencode 中使用
