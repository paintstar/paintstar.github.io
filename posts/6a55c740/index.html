解决服务器中使用 codex 网络问题

codex 使用需要外网，但是因为服务器一般没法连外网，所以需要通过 ssh 转发到本地。看到网上一堆一堆的设置，决定自己写一个简单版本的。

假设本地 vpn 部署在 7897 端口，服务器使用 KeyPair-f56e.pem 密钥登录，用户为 ma-user，端口为 32558。

打开 cmd 使用 以下命令进行 ssh 转发，注意这个 cmd 窗口不能关闭：

ssh -i ~/KeyPair-f56e.pem -N -R 7897:127.0.0.1:7897 ma-user@syai-2 -p 32558

这样服务器到 7897 端口的请求都会到 本机 7897 。
